### What is HTML?

HTML (**HyperText Markup Language**) is the standard markup language used to create and structure web pages. It defines the layout and content of a webpage using different elements called **HTML tags**. HTML is used to organize content such as headings, paragraphs, images, links, tables, lists, and other webpage components.

---

## HTML Tags

HTML tags are the basic building blocks of an HTML document. They are keywords enclosed within angle brackets (`< >`) and are used to define different elements on a webpage. Most HTML tags consist of an **opening tag** and a **closing tag**, with the content placed between them.

### Common HTML Tags

| Tag | Description |
|------|-------------|
| `<!DOCTYPE html>` | Declares the document as an HTML5 document. |
| `<html>` | Root element that contains the entire HTML document. |
| `<head>` | Contains metadata such as title, styles, and links. |
| `<title>` | Displays the webpage title in the browser tab. |
| `<body>` | Contains all the visible content of the webpage. |
| `<h1>` | Largest heading used for the main title. |
| `<h2>` | Second-level heading for major sections. |
| `<h3>` | Third-level heading for sub-sections. |
| `<h4>` | Fourth-level heading for smaller sections. |
| `<h5>` | Fifth-level heading for minor headings. |
| `<h6>` | Smallest heading level. |
| `<p>` | Defines a paragraph of text. |
| `<br>` | Inserts a line break. |

---

## HTML Table

An **HTML table** is used to display data in a structured format consisting of rows and columns. Tables are useful for presenting information such as marks, timetables, schedules, price lists, and reports.

A table is created using the `<table>` element and contains:
- `<tr>` – Defines a table row.
- `<th>` – Defines a table heading.
- `<td>` – Defines a table data cell.

### Key Points
- Organizes data in rows and columns.
- Improves readability of structured information.
- Commonly used for tabular data like reports and schedules.

---

## Colspan and Rowspan

`colspan` and `rowspan` are attributes used to merge table cells.

### Colspan
- Merges two or more columns into a single cell.
- Used when one cell needs to span horizontally across multiple columns.

### Rowspan
- Merges two or more rows into a single cell.
- Used when one cell needs to span vertically across multiple rows.

### Benefits
- Creates cleaner and more organized table layouts.
- Helps design complex tables with merged cells.
- Improves readability and presentation of data.
