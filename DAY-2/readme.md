## HTML Lists

HTML lists are used to organize and display related information in a structured format. They improve readability and make content easier to understand.

### Types of HTML Lists

### Unordered List (`<ul>`)
- Displays items using bullet points.
- Used when the order of items is not important.

### Ordered List (`<ol>`)
- Displays items in a numbered or alphabetical sequence.
- Used when the order of items is important.

### List Item (`<li>`)
- Represents an individual item within an ordered or unordered list.

### Benefits of HTML Lists
- Organize information in a structured manner.
- Improve the readability of webpages.
- Make content easier to understand and navigate.

## Marquee

The **`<marquee>`** element is used to create scrolling text or images on a webpage. The content can move horizontally or vertically in different directions, such as left, right, up, or down.

### Key Points
- Creates scrolling text or images.
- Supports different scrolling directions.
- Considered an outdated HTML element and is not recommended for modern web development.

---

## Image Tag

The **`<img>`** tag is used to display images on a webpage. It is a self-closing tag and requires the image source to display the desired image.

### Key Points
- Displays images on a webpage.
- Self-closing HTML element.
- Supports alternative text for accessibility.
- Commonly used to enhance the visual appearance of websites.

---

## Video Tag

The **`<video>`** tag is used to embed video files directly into a webpage, allowing users to play videos without requiring external plugins.

### Key Points
- Embeds videos in a webpage.
- Supports playback controls such as play, pause, and volume.
- Can display videos in different supported formats.
- Improves user engagement by adding multimedia content.
