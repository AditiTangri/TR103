# Calculator using HTML, CSS & JavaScript

## Overview

This project is a simple calculator built using **HTML**, **CSS**, and **JavaScript**. It performs the four basic arithmetic operations:

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)

The calculator uses the **onclick** event to detect button clicks and the **.value** property to read and display values in the input field.

---


## Concepts Used

### onclick Event
The `onclick` event is used to call a JavaScript function whenever a button is clicked.

**Syntax**
```html
<button onclick="functionName()">Click Me</button>
```

### .value Property
The `.value` property is used to get or set the value of an input element.

**Get Value**
```javascript
element.value;
```

**Set Value**
```javascript
element.value = "new value";
```

---

## Functions Used

### addvalue()
Adds the clicked number or operator to the calculator display.

### cleard()
Clears the calculator display.

### eva()
Reads the expression entered by the user, identifies the operator, performs the calculation, and displays the result.
