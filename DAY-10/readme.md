## Switch Statement
A switch statement is a selection control structure used to execute one block of code among many based on the value of an expression.

### Key Points:
- Compares a variable with multiple case values  
- Executes the matching case block  
- `break` is used to exit the switch  
- `default` runs if no case matches  

---

## Loops
A loop is a control structure used to execute a block of code repeatedly as long as a condition is true. It helps avoid writing repeated code.

### Types of Loops:

### 1. for loop
Used when the number of repetitions is known in advance.
- Initialization, condition, and update are written in one line  
- Best for counting tasks  

---

### 2. while loop
Used when the number of repetitions is not fixed.
- Condition is checked first  
- Runs only if condition is true  
- Repeats until condition becomes false  

---

### 3. do-while loop
- Executes the block at least once  
- Condition is checked after execution  

---

## break Statement
The `break` statement is used to immediately terminate a loop or switch.

### Uses:
- Exits the entire loop  
- Exits the switch block  

---

## continue Statement
The `continue` statement skips the current iteration of a loop and moves to the next iteration.

### Key Points:
- Does NOT stop the loop  
- Skips only current iteration  

---

## Functions
A function (method in Java) is a reusable block of code that performs a specific task.

### Benefits:
- Reduces repetition  
- Improves modularity  
- Can take inputs and return outputs  

---

## DOM (Document Object Model)
DOM is a programming interface for HTML/XML used mainly in JavaScript.

### Key Points:
- Represents webpage as a tree structure  
- Each HTML element is a node  
- Allows dynamic changes to content, structure, and style
