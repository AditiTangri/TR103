## JAVASCRIPT

JavaScript is a versatile, dynamically typed programming language that brings life to web pages by making them interactive. It is used for building interactive web applications and supports both client-side and server-side development.

### Features:
- **Interpreted language:** Code is executed line by line.  
- **Dynamically typed:** Variable types are determined at runtime.  
- **Single-threaded:** Executes one task at a time but supports asynchronous operations.  

---

## Adding JavaScript in an HTML Document

JavaScript is used to make web pages interactive and dynamic. It is added to an HTML document using the `<script>` tag.

### Ways to add JavaScript:
1. **Inline JavaScript** – Written directly inside HTML elements using event attributes like `onclick`.  
2. **Internal JavaScript** – Written inside the `<script>` tag within the HTML file (in `<head>` or `<body>`).  
3. **External JavaScript** – Written in a separate `.js` file and linked using `<script src="script.js"></script>`.  

### Advantages of External JavaScript:
- Improves code organization and readability  
- Allows code reuse across multiple pages  
- Makes maintenance easier  
- Improves page loading speed through caching

## JavaScript Variables

JavaScript variables are used to store data values. They can be declared using `var`, `let`, or `const`.

### Types:
- **var** – Function-scoped and can be redeclared (older JavaScript).  
- **let** – Block-scoped and can be updated but not redeclared.  
- **const** – Block-scoped and cannot be reassigned.  

---

## JavaScript Operators

JavaScript operators are symbols used to perform operations on variables and values.

### Types of Operators:
- **Arithmetic Operators:** `+`, `-`, `*`, `/`, `%`  
- **Assignment Operators:** `=`, `+=`, `-=`, `*=`, `/=`  
- **Comparison Operators:** `==`, `===`, `!=`, `>`, `<`, `>=`, `<=`  
- **Logical Operators:** `&&`, `||`, `!`  
- **Unary Operators:** `++`, `--`, `typeof`, `delete`  
- **String Operators:** `+`, `+=`  

---

## Conditional Statements in JavaScript

Conditional statements are used to make decisions in a program based on conditions.

### Types:
- **if statement:** Executes code if condition is true.  
- **if...else statement:** Executes one block if true, otherwise another block.  
- **if...else if...else statement:** Used for multiple conditions.  
- **switch statement:** Compares a value with multiple cases.  

---

## parseInt() in JavaScript

- Converts a string into an integer (whole number).  
- Ignores decimal values and keeps only integer part.  
- Returns `NaN` if the value is not a number.
