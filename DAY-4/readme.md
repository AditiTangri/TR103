# Introduction to CSS

**CSS (Cascading Style Sheets)** is a stylesheet language used to control the appearance and layout of HTML webpages. It allows developers to add colors, fonts, spacing, borders, animations, and responsive designs, making webpages visually attractive and user-friendly.

---

# Types of CSS

CSS can be applied to HTML documents in three different ways:

## 1. Inline CSS

Inline CSS is written directly inside an HTML element using the **style** attribute.

### Features
- Applies styling to a single HTML element.
- Has the highest priority among all CSS types.
- Useful for quick styling or testing small changes.
- Difficult to maintain in large projects because styling is mixed with HTML.

---

## 2. Internal CSS

Internal CSS is written within the **`<style>`** tag inside the **`<head>`** section of an HTML document.

### Features
- Styles only the current webpage.
- Keeps styling separate from the HTML content while remaining in the same file.
- Easier to manage than inline CSS.
- Cannot be reused across multiple webpages.

---

## 3. External CSS

External CSS is written in a separate file with the **`.css`** extension and linked to an HTML document.

### Features
- Can be used to style multiple webpages.
- Makes code clean, organized, and easier to maintain.
- Any change made in the CSS file automatically updates all linked pages.
- It is the most commonly used and recommended method in professional web development.

---

# CSS Box Model

The **CSS Box Model** explains how every HTML element is displayed as a rectangular box. It helps control the size, spacing, and layout of elements on a webpage.

Every box consists of four main parts:

### 1. Content
- The area where text, images, or other HTML elements are displayed.
- It forms the main part of the element.

### 2. Padding
- The space between the content and the border.
- It creates internal spacing, making the content appear away from the border.

### 3. Border
- The visible outline surrounding the content and padding.
- Borders can have different widths, styles, and colors.

### 4. Margin
- The space outside the border.
- It creates distance between one element and another, improving the overall layout.
