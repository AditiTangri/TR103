# setInterval()

`setInterval()` is used to execute a function repeatedly after a specified time interval (in milliseconds). It continues running until it is stopped using `clearInterval()`.

### Syntax

```javascript
let intervalId = setInterval(function, delay);
```

### Parameters
- function – Function to execute repeatedly.
- delay – Time interval in milliseconds(1000 ms = 1 second)

---

# clearInterval()

clearInterval() stops the repeated execution started by setInterval().

### Syntax

```javascript
clearInterval(intervalId);
```

# JavaScript Date Object

The JavaScript Date object is used to work with dates and times. It can create, display, and manipulate date and time values.

### Creating a Date
#### Current Date and Time
```javascript
let today = new Date();
```
#### Specific Date
```javascript
let d = new Date("2026-07-10");
```
#### Using Year, Month, Day
```javascript
let d = new Date(2026, 6, 10);
```
