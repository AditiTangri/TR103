## HTML Form

An **HTML form** is used to collect user input and send it to a server for processing. Forms allow users to enter information such as names, passwords, email addresses, and other details.

### Common Form Attributes

- **action** – Specifies the URL where the form data will be sent after submission.
- **method** – Defines how the form data is sent to the server. The two commonly used methods are:
  - **GET** – Sends data through the URL. Suitable for non-sensitive information.
  - **POST** – Sends data securely in the request body. Used for sensitive or large amounts of data.

# Common HTML Input Types

HTML provides different input types to collect various kinds of user information.

## 1. Text & Information Inputs

### Text
- Used to enter a single line of text.
- Commonly used for names, usernames, and short text.

### Password
- Hides the entered characters using dots or asterisks.
- Used for secure password entry.

### Email
- Accepts only valid email address formats.
- Provides built-in browser validation.

---

## 2. Selection & Choice Inputs

### Checkbox
- Allows users to select one or more options.
- Multiple checkboxes can be selected simultaneously.

### Radio Button
- Allows users to select only one option from a group.
- All radio buttons in the same group share the same name.

### Color
- Opens the system's color picker.
- Used to select a color value.

### File
- Allows users to browse and upload files such as documents or images.

---

## 3. Action Buttons

### Submit
- Sends the completed form data to the server.
- Executes the form's action.

### Reset
- Restores all form fields to their default values.
- Clears any changes made by the user.

### Button
- Creates a standard clickable button.
- Commonly used with JavaScript to perform custom actions.

### Image
- Uses an image instead of a regular button.
- Clicking the image submits the form.

---

## 4. Number & Range Inputs

### Number
- Accepts only numeric values.
- Can include minimum, maximum, and step values.

### Range
- Displays a slider for selecting a value within a specified range.
- Useful for approximate numerical input.

---

## 5. Date & Time Pickers

### Date
- Displays a calendar for selecting a date.
- Allows users to choose the day, month, and year.

### Time
- Displays a time picker.
- Used to select hours and minutes.
