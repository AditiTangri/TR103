## Bootstrap Introduction

Bootstrap is a popular front-end framework used to develop responsive and mobile-friendly websites quickly.

It provides ready-made CSS classes and components that help developers create modern user interfaces with less custom CSS.

Bootstrap follows a mobile-first approach, which means designs are created for smaller screens first and then adjusted for larger devices.

Bootstrap can be included in a project using:

- Bootstrap CDN
- Downloaded Bootstrap files

---

## Bootstrap Containers

Containers are the basic layout components in Bootstrap. They are used to center content and provide proper spacing around webpage elements.

Bootstrap provides two main types of containers:

### `.container`

A responsive fixed-width container.

The width automatically changes according to different screen sizes.

It is commonly used for creating centered layouts.

### `.container-fluid`

A full-width container that occupies 100% of the available screen width.

It is useful for creating layouts that cover the entire viewport.

---

## Bootstrap Row

A row is used to create a horizontal group of columns inside a container.

Rows are an important part of the Bootstrap Grid System because they organize columns properly and maintain alignment.

The row divides the webpage structure into horizontal sections.

---

## Bootstrap Grid System

Bootstrap uses a 12-column responsive grid system.

The main parts of the grid system are:

- Container
- Row
- Columns

Each row contains 12 equal columns. Developers can decide how many columns an element should occupy according to the required layout.

The grid system helps create responsive designs for different devices.

---

## Bootstrap Columns

Columns are used to divide content inside a row.

Bootstrap provides different column classes to control the width of elements.

Examples of column classes:

- `.col`
- `.col-1`
- `.col-3`
- `.col-6`
- `.col-md-6`
- `.col-lg-4`

### `.col-1`

Occupies 1 column out of 12 columns.

### `.col-3`

Occupies 3 columns out of 12 columns.

---

## 6. Bootstrap Breakpoints

Breakpoints are predefined screen sizes where Bootstrap changes the layout according to device width.

They help create responsive websites that work properly on different devices.

Bootstrap breakpoints include:

| Breakpoint | Screen Size |
|------------|-------------|
| xs | Less than 576px |
| sm | 576px and above |
| md | 768px and above |
| lg | 992px and above |
| xl | 1200px and above |
| xxl | 1400px and above |

---

## Responsive Grid Classes

Bootstrap provides responsive grid classes that apply different column sizes according to screen size.

Examples:

### `.col-md-6`

- Works on medium screens and above.
- Occupies 6 columns out of 12.

### `.col-lg-4`

- Works on large screens and above.
- Occupies 4 columns out of 12.

Responsive classes help create layouts that automatically adjust for mobile, tablet, and desktop screens.

---

## Bootstrap Margin Utilities

Margin creates space outside an element.

Bootstrap provides predefined margin classes to control spacing without writing custom CSS.

Margin utilities can be applied to:

- All sides
- Top
- Bottom
- Start
- End
- Horizontal direction
- Vertical direction

Margin helps maintain proper distance between different webpage elements.

Common margin classes:

- `m`
- `mt`
- `mb`
- `ms`
- `me`
- `mx`
- `my`

Margin values range from:

- `0`
- `1`
- `2`
- `3`
- `4`
- `5`

---

## Bootstrap Padding Utilities

Padding is the space inside an element between the content and the element’s border.

Bootstrap provides padding utility classes to add internal spacing easily.

Padding can be applied to:

- All sides
- Top
- Bottom
- Start
- End
- Horizontal direction
- Vertical direction

Padding improves the appearance and readability of content inside elements like cards, buttons, containers, and sections.

Common padding classes:

- `p`
- `pt`
- `pb`
- `ps`
- `pe`
- `px`
- `py`
